# chen-ui

## 前言

* chen-ui是一个有趣的ui组件库，包含了一些开发中可能会经常用到的组件，也包含一些个人觉得有趣但不是很常用的组件。
* 目前该组件库还没有成型。如果有想使用该组件库的一些组件，可以到该组件的目录下复制该组件代码。后期如果组件多起来的话会考虑发布到npm上供大家使用。
* 设计想法：区别于常见的ui组件库，想支持更多的自定义ui样式，例如颜色、大小等。
* 该组件库会不定时更新一些组件（主人很懒）。
* 欢迎有兴趣的小伙伴加入。有好的意见也欢迎提出来。
